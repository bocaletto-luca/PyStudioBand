PyStudioBand

Manage audio & music production applications on Debian & Ubuntu with ease

Get Started

Overview

PyStudioBand is a modern GTK3-based desktop application that simplifies the management of audio and music-production software on Debian and Ubuntu. From DAWs to synthesizers, PyStudioBand offers an intuitive interface to install, remove, launch and organize your favorite tools.

Key Features

  • Curated catalog of popular audio and music-production applications
  • One-click install, remove or purge via apt-get
  • Real-time status of installed vs. available software
  • Launch multiple applications simultaneously
  • Organize by category: DAW, Editor, Server, Synthesizer, Plugin Host, Looper, and more
  • Add, edit or delete custom entries in your personal catalog
  • Integrated Help & Documentation panel

How It Works

  1. Load Catalog: PyStudioBand merges a built-in list of essential apps with any user-added entries stored in ~/.pystudiomusic/apps.custom.
  2. Check Status: The application queries dpkg-query to determine which packages are installed.
  3. Install or Remove: Select desired apps and click “Apply Changes” to install, remove or purge via apt-get under the hood.
  4. Launch Tools: Switch to the Launch tab, tick the apps you want, and launch them in parallel.
  5. Customize Catalog: Add new software definitions with a friendly form (ID, name, category, description, pkg, command).

Installation

git clone https://github.com/yourusername/PyStudioBand.git
cd PyStudioBand
sudo apt update
sudo apt install -y python3-gi python3-gi-cairo gir1.2-gtk-3.0
chmod +x PyStudioBand.py
./PyStudioBand.py

Optionally, install a desktop entry: 

sudo cp data/PyStudioBand.desktop /usr/share/applications/
update-desktop-database

Usage

Launch PyStudioBand from your terminal or applications menu. Navigate through the sidebar to manage applications, view status, launch tools, add custom entries, or read detailed help.

Customization

Your custom definitions live in ~/.pystudiomusic/apps.custom. You can edit this plain-text file directly or use the “Add App” form. The format is: 

uid|Name|Category|Description|apt-package|launch-command
